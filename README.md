# Homebrew Tap for tap

## Add this tap to your Homebrew:

```sh
brew tap zjt003/tap
```

## Then install the apps you want:

- v2rayn:
```sh
brew install --cask v2rayn
```

- mihomo-party:
```sh
brew install --cask mihomo-party
```
